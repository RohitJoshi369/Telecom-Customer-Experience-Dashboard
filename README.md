**Overview**
This project analyzes customer service interactions for a telecom company to uncover insights into customer satisfaction (CSAT), agent performance, and call center operations.
The goal is to identify key drivers of customer experience and provide actionable recommendations to improve service quality and operational efficiency.

**Dataset Description**
The dataset includes three main components:

**Account Interactions**
Call Date, Call Reason, Call Resolution
**Agent Information**
Agent ID, Role (Tier 1 / Tier 2), Tenure

**Customer Satisfaction Results**
Ratings (1–5) for:
Overall Experience
Agent Quality
Issue Resolution
Courtesy
Knowledge

**Analyze customer call patterns and reasons**
Evaluate agent performance across roles and tenure
Identify factors impacting customer satisfaction
Provide data-driven recommendations

**Tools & Technologies**
Power BI
Excel / CSV (data source)
SQL (data querying)
